Page Components is a small collection of CSS components and tweaks designed specifically for Itch.io project pages.

Get started with the documentation
  • Responsive layout adapts to different screen sizes
  • Automatically match your page's theme
  • Free and open source under CC0 license

Live Preview

The components make use of your page's theme. Interract with the components below. Hover over to the different colors to change the theme. You can resize the preview window by dragging the bottom-right corner.

Can I use this in my commercial project?Yes! All the HTML & CSS codes here is licensed under Creative Commons Zero v1.0 Universal
Whats the difference between the files?components.alt.css contains all components. Allows you to delete unused components, and choose the one you want.

components.min.css is the fully compressed version. With all components and smaller size.

components.css is the prettified version. Allows you to modify the CSS codes easily.
