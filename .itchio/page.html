Spice up and level up your page!

Pitch is a small collection of CSS components and tweaks designed specifically for Itch.io project pages.

Get started with the documentation
  • Responsive layout adapts to different screen sizes
  • Automatically match your page's theme
  • Free and open source under CC0 license

Live Preview

The components make use of your page's theme. Interract with the components below. Hover over to the different colors to change the theme. You can resize the preview window by dragging the bottom-right corner.

How do I get access to custom CSS in my game page? You can contact the itch.io support to enable the custom CSS feature.
Can I use this in a commercial project? Yes! All the HTML & CSS codes here is licensed under Creative Commons Zero v1.0 Universal
The components didn't appear correctly in my page, what should I do? Submit a new Issue at the GitHub Repository for bug report. Provides your HTML/CSS codes, description of what are you trying to achieve, and the browser/app you were on to view your page.
