How do I get access to custom CSS in my game page? You can contact the itch.io support to enable the custom CSS feature.
Can I use this in a commercial project? Yes! All the HTML & CSS codes here is licensed under Creative Commons Zero v1.0 Universal.
Why can't I use this on profile or jam pages? All components make use of the page's theme. Unlike project page, profile page and jam page didn't have some of the required color variables needed for the components to be displayed correctly (e.g. button color).
