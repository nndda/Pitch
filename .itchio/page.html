Requirements

Custom CSS Access, & a little bit of HTML + CSS knowledge. You can request CSS access on your account via itch.io support.

Installation

Download and extract the files, copy the content of components.alt.css to your page's custom CSS.
Edit your project description in HTML mode, and follow the usage guide for each of the components in the documentation.

Accordion

How do I get access to custom CSS in my game page?You can contact the itch.io support to enable the custom CSS feature.
Can I use this in my commercial project?Yes! All the HTML & CSS codes here is licensed under Creative Commons Zero v1.0 Universal
Whats the difference between the files?components.alt.css contains all components. Allows you to delete unused components, and choose the one you want.

components.min.css is the fully compressed version. With all components and smaller size.

components.css is the prettified version. Allows you to modify the CSS codes easily.

Inputs

  • Press CTRL + C to copy a text
  • Hold down RMB to aim, Press LMB to shoot
